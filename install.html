Install

This section contains the instructions to install bioimageit_gui.

Note

Installers for stable realeases are available in the BioImageIT website. We recommand using these installer for BioImageIT users.

If you are a developer, you can use the instructions below to get the under developement main version of BioImageIT.

Install script for developers

BioImageIT GUI depends on several BioImageIT project libraries (bioimageit_formats, bioimageit_core, bioimageit_viewer…) and the data processing toolboxes. The install process can then be tedious. This is the reason why we provide install scripts for each operating systems:

Windows

Run the executable <https://github.com/bioimageit/bioimageit-install/raw/v0.2.0/windows/Install_BioImageIT.exe>.

MacOS

Run the MacOS install script The install can also be done using the install application available as a dmg

Linux

Run the linux install scipt.
